Python2.x
---------

python2.x中输出默认是换行的，为了抑制换行，可以在打印最后加一个逗号

Python3.x
---------

到了python3中，print 变成一个函数，这种语法便行不通了。  
我们可以使用  print(x, end="")   
end="" 可使输出不换行。双引号之间的内容就是结束的内容， 可以是空格，也可以是其他字符，默认为换行
